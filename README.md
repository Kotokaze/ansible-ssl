# Ansible for Robocup SSL
## Usage
1. Install [Ansible](https://www.ansible.com/) with `sudo apt install ansible`
2. Clone this repository
3. Edit [hosts](hosts) file and add your hosts
4. Run `ansible-playbook -i hosts site.yml`
